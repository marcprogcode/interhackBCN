# Interhack BCN Retention API Documentation

The Retention API provides a real-time interface to fetch prioritized client alerts generated by the analytical engine. It helps sales teams identify which clients to contact first to maximize retention and revenue.

## Base URL
By default, the server runs on:
`http://localhost:8000`

## Interactive Documentation
Once the server is running, you can access interactive API documentation at:
- **Swagger UI**: [http://localhost:8000/docs](http://localhost:8000/docs)
- **ReDoc**: [http://localhost:8000/redoc](http://localhost:8000/redoc)

---

## Endpoints

### 1. Fetch Prioritized Alerts
Retrieves a list of high-priority client alerts, enriched with location data and normalized scores.

**URL**: `/alerts`  
**Method**: `GET`  
**Auth Required**: No

#### Query Parameters
| Parameter | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `top_x` | `integer` | `10` | The number of highest-priority alerts to return. |

#### Success Response
**Code**: `200 OK`  
**Content**: `Array of Objects`

| Field | Type | Description |
| :--- | :--- | :--- |
| `company_id` | `string` | Unique identifier of the client company. |
| `location` | `string` | Province where the company is located. |
| `product_family` | `string` | The product family the alert applies to. |
| `type` | `string` | The type of alert (e.g., Replenishment, Churn Risk). |
| `reason` | `string` | Human-readable explanation of why the alert was triggered. |
| `priority_score` | `float` | Normalized score from **1.0 to 10.0** within the current selection (10 = highest priority). |
| `expected_return` | `float` | Estimated revenue (in €) from a successful outreach. |
| `confidence` | `float` | Score from **0.0 to 1.0** indicating the reliability of the purchase pattern. |

#### Example Request
```bash
curl "http://localhost:8000/alerts?top_x=3"
```

## Technical Details

### Server Execution
To start the API server:
```bash
python src/api.py
```

### Dependencies
The API requires the following Python packages:
- `fastapi`
- `uvicorn`
- `pandas`
- `numpy`
